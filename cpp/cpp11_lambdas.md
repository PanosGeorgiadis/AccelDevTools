## **C++11 lambdas**

lambda expression
(function) closure: lexically scoped name binding, where functions are first-class

### Syntax

1. `[captures]<tparams> (params) specifiers exception attr -> ret requires`
2. `[captures](params) -> ret { body }`
3. `[captures](params) { body }`
4. `[captures] { body }`
